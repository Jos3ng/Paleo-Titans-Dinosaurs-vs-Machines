# Paleo-Titans-Dinosaurs-vs-Machines
Paleo Titans é um RPG incremental/idle onde você escava fósseis, revive dinossauros elementais, monta equipes de até 5 criaturas e enfrenta hordas de robôs. Colecione, evolua, descubra raridades, crie estratégias elementais e avance o máximo possível em um sistema roguelike de progressão infinita.
